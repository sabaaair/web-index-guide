# web-index-guide

## 项目简介

`web-index-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，主要目的是集中存放各类静态页面资源，方便检索与引用。

## 目录说明

```
web-index-guide/
├── pages/          # 存放所有归档的 HTML 页面文件
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── index.html      # 仓库首页导航（可选）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件代表一个独立归档页面，命名清晰，便于区分。
- `assets/`：按需组织，建议按页面或功能分类存放。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接通过浏览器打开预览。
- 页面之间不相互依赖，可单独部署或引用。
- 归档内容通常为历史快照、参考模板或实验性页面，不保证长期更新。

## 维护说明

- 欢迎提交 Pull Request 添加新的归档页面，请确保页面内容合规且不包含敏感信息。
- 如需更新已有页面，建议保留旧版本并新增文件，避免直接覆盖。
- 仓库维护者会不定期清理无效或过时页面，但会尽量保留历史记录。
